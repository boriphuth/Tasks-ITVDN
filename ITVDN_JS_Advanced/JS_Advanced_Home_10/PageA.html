HTTP (Hyper Text Transfer Protocol) – протокол, который определяет как веб браузер должен запрашивать документы, как информация должна передаваться серверам и как веб-сервера должны отвечать на веб запросы. В первую очередь с протоколом HTTP работает веб браузер, в тоже время с HTTP протоколом может работать JavaScript сценарий. HTTP запросы могут инициироваться, когда сценарий устанавливает значение свойства location или вызывает submit объекта form.
